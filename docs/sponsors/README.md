# 赞助商 Banner

本目录存放 README 赞助商板块使用的品牌图片。

## 约定

- 文件名用小写英文 + 连字符，例如 `ztest.png`、`acme-relay.png`
- 建议使用 **宽度 300px 左右** 的横版 logo / banner，PNG 或 JPG 均可
- 在根目录 `README.md` 的赞助商表格中以相对路径引用，例如：
  ```html
  <td width="180"><a href="链接"><img src="docs/sponsors/ztest.png" alt="名称" width="150"></a></td>
  ```

## 现有文件

- `ztest.png` —— 真测 Ztest（当前为 app 图标占位，建议替换为真测品牌 banner）
